## 部署步骤

### 1️⃣ 上传文件到服务器
```bash
scp -r ./中间件docker-compse-生产环境部署包 user@115.191.79.224:/opt/mall4cloud/
```

### 2️⃣ SSH连接服务器
```bash
ssh user@115.191.79.224
cd /opt/mall4cloud/中间件docker-compse-生产环境部署包
```

### 3️⃣ 设置权限
```bash
chmod -R 777 ./rocketmq
chmod -R 666 ./minio/data
chmod -R 777 ./elasticsearch/data 2>/dev/null || true
```

### 4️⃣ 启动服务
```bash
docker compose up -d --build
```

### 5️⃣ 等待服务启动（约5-10分钟）
```bash
# 查看状态
docker compose ps

# 查看日志
docker compose logs -f | grep -v nginx
```

## 常见问题快速诊断

### 问题：容器启动失败
```bash
# 查看详细日志
docker compose logs <service-name>

# 检查端口占用
netstat -tlnp | grep <port>
```

### 问题：MySQL连接失败
```bash
# 检查MySQL日志
docker compose logs mysql

# 检查binlog状态
docker exec mall4cloud-mysql mysql -u root -pGesoft9919. -e "SHOW MASTER STATUS;"
```

### 问题：RocketMQ启动失败
```bash
# 检查Namesrv日志
docker compose logs rocketmq-namesrv

# 检查Broker日志
docker compose logs rocketmq-broker

netstat -tlnp | grep 9876
```
