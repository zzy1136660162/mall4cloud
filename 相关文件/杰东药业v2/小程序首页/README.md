# 杰东药业小程序首页

## 项目结构

```
小程序首页/
├── app.js                 # 小程序入口JS
├── app.json               # 小程序全局配置
├── app.wxss               # 小程序全局样式
├── pages/
│   └── index/
│       ├── index.wxml     # 首页结构
│       ├── index.wxss     # 首页样式
│       ├── index.js       # 首页逻辑
│       └── index.json     # 首页配置
└── images/                # 图片资源目录
```

## 设计说明

### 主色调
- **主色**: `#134585` (深蓝色)
- **辅助色**: `#1a5aa0` (亮蓝色)
- **背景色**: `#f5f7fa` (浅灰蓝)

### 页面结构

1. **顶部Banner**
   - 企业口号：共创医健产业创新生态圈
   - 行动纲领：大道至简 · 实干为要

2. **核心数据展示**
   - 13年行业深耕
   - 3万+服务企业
   - 1000+精选产品

3. **三个切换按钮**（核心交互）
   - 🏆 杰东认证
   - 🏭 杰东生产
   - 🚀 杰东营销

4. **内容区域**（根据切换显示不同内容）

### 三大板块内容

#### 1. 认证板块
- 6大认证类别卡片（医疗器械、化妆品、消毒用品、保健用品、功能性食品、院内制剂）
- 服务优势列表
- 立即咨询认证按钮

#### 2. 生产板块
- 剂型类型标签（膏剂、粉剂、片剂、液体、颗粒、凝胶）
- 产品类别网格
- 15大保健用品功效标签
- 生产优势列表
- 获取贴牌方案按钮

#### 3. 营销板块
- 核心模式卡片（上游整合、中台枢纽、下游裂变）
- 资源优势数据卡片
- 货盘矩阵展示
- 特色产品列表
- 浏览产品库/成为合伙人按钮

## 使用说明

1. 点击顶部三个按钮可切换不同板块内容
2. 内容区域支持滚动查看
3. 底部有固定导航栏
4. 支持下拉刷新
