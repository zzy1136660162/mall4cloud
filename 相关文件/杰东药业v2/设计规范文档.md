## 六、各模块设计标准

### 6.1 服务优势模块
- **布局**：2列网格，gap: 10px
- **卡片**：白色背景，圆角10px，居中文字
- **数字**：22px，金色 #cda360，加粗
- **标题**：13px，#333，加粗
- **描述**：11px，#666

### 6.2 生产优势模块
- **外框**：灰色背景 #f5f7fa，圆角16px，内边距12px
- **布局**：绝对定位，4张图片错落分布
  - 左上：宽 calc(50%-18px)，高150px
  - 右上：宽 calc(50%-18px)，高90px
  - 左下：宽 calc(50%-18px)，高90px
  - 右下：宽 calc(50%-18px)，高150px
- **图片**：圆角12px，object-fit: cover
- **遮罩**：线性渐变 transparent 0% → rgba(25,66,118,0.5) 50% → rgba(25,66,118,0.92) 100%
- **数字**：18px，金色 #cda360，带阴影
- **文字**：11px，白色

### 6.3 资源优势模块
- **布局**：3列网格，gap: 10px
- **卡片**：金黄渐变背景，圆角8px，内边距12px 10px
- **渐变**：#ebc880 → #d0a860 → #b8944f
- **数字**：22px，白色，加粗，带阴影
- **标题**：12px，白色，加粗
- **阴影**：0 4px 12px rgba(205,163,96,0.25)

### 6.4 导航标签
- **默认**：背景 #f5f7fa，文字 #555
- **激活**：背景 #ffffff，文字 #194276，底部边框 #194276
- **圆角**：12px 12px 0 0
- **内边距**：12px 20px 10px

### 6.5 按钮规范
- **主按钮**：背景 #194276，文字白色，圆角8px
- **次按钮**：背景白色，边框 #194276，文字 #194276
- **内边距**：12px 20px
- **字号**：14px

## 七、响应式原则

- 容器最大宽度：100%（移动端全宽）
- 水平边距：20px
- 图片自适应：object-fit: cover
- 网格自适应：flex-wrap / grid 自动换行

---

## 八、动效规范

| 场景 | 效果 | 时长 |
|-----|------|------|
| 卡片悬停 | 阴影加深 + 边框变色 | 0.3s ease |
| 按钮悬停 | 背景色变化 | 0.25s |
| 标签切换 | 底部边框滑动 | 0.3s |
| 图片悬停 | 轻微上浮 | 0.3s ease |

- 尺寸：20-24px
- 描边：1.5-2px
- 颜色：#194276（深色背景用白色）
- 风格：线性图标，无填充
