# 商品商城搜索功能 - 完整说明文档

## 📦 功能概述

实现了一套完整的淘宝风格搜索功能，包含搜索页面、搜索结果页面和商城首页，所有文件均以`sp`作为前缀命名。

## 🎯 功能流程

### 1. 搜索交互流程

```
商城首页 → 点击搜索框 → 搜索页面 → 输入关键词 → 点击搜索
    ↓
搜索结果页面（单列卡片布局）→ 点击商品 → 商品详情页
```

### 2. 搜索页面功能 (sp_SearchPage)

**核心功能：**
- ✅ 点击搜索框跳转到专用搜索页面
- ✅ 醒目的搜索输入框和搜索按钮
- ✅ 300ms防抖搜索建议
- ✅ 搜索历史记录（本地存储，最多20条）
- ✅ 热门搜索推荐
- ✅ 点击建议、历史、热搜直接跳转结果

**交互特点：**
- 输入时实时显示搜索建议
- 点击建议项直接跳转搜索结果
- 点击热门搜索标签快速搜索
- 点击历史记录重复搜索
- 支持删除单条历史和清空全部历史

### 3. 搜索结果页面功能 (sp_SearchResult)

**布局规范：**
- ✅ 卡片式设计
- ✅ **单列布局**（一列一个商品卡片）
- ✅ 垂直排列
- ✅ 每个卡片包含：
  - 商品图片
  - 商品名称
  - 价格信息
  - 销量信息
  - 标签
  - 操作按钮

**卡片信息：**
```

│                             │
├─────────────────────────────┤

├─────────────────────────────┤

├─────────────────────────────┤
│ [+加入购物车] [立即购买]     │

```

### 4. 商城首页功能 (sp_Mall)

**功能特点：**
- ✅ 顶部搜索框（点击跳转到搜索页面）
- ✅ 热销爆款（横向滚动）
- ✅ 新品上市（横向滚动）
- ✅ 为你推荐（横向滚动）
- ✅ 悬浮购物车按钮
- ✅ 快速加入购物车

## 🔧 技术实现

### 防抖搜索
```javascript
// 300ms 防抖
onInput(e) {
  clearTimeout(this.searchTimer)
  this.searchTimer = setTimeout(() => {
    this.loadSuggestions(value)
  }, 300)
}
```

### 历史记录存储
```javascript
// 本地存储
wx.setStorageSync('sp_search_history', list)

// 读取
wx.getStorageSync('sp_search_history') || []
```

### API调用
```javascript
// 搜索商品
productApi.searchProducts(keyword, page, pageSize)
```
