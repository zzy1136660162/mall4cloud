## 🎯 开发任务详情

### 任务1.1：创建API文件
**文件:** `src/api/modules/selection.ts`

**内容:**
- 导入API基础模块
- 定义类型接口（SelectionApplyVO、SelectionApplyPageParam等）
- 导出API方法（page、audit、ship、statistics）

**代码示例:**
```typescript
import api from '@/api'

export interface SelectionApplyVO {
  applyId: number
  userName: string
  contactPhone: string
  // ... 其他字段
}

export const selectionApi = {
  page: (params) => api.request({ url: '/mall4cloud_platform/admin/selection/applies', method: 'get', params }),
  audit: (applyId, data) => api.request({ url: `/mall4cloud_platform/admin/selection/audit/${applyId}`, method: 'put', data }),
  ship: (applyId, expressCompany, expressNo) => api.request({ url: `/mall4cloud_platform/admin/selection/ship/${applyId}`, method: 'put', params: { expressCompany, expressNo } }),
  statistics: () => api.request({ url: '/mall4cloud_platform/admin/selection/statistics', method: 'get' }),
}
```

## 🐛 常见问题

### Q1: 如何处理价格显示？
**A:** `priceFee` 单位为分，需要除以100转换：
```typescript
{{ (detail.priceFee / 100).toFixed(2) }}  // 显示为 "99.99"
```

### Q2: 审核状态如何显示？
**A:** 使用 `el-tag` 组件：
```vue
<el-tag :type="getStatusType(detail.auditStatus)">
  {{ detail.auditStatusText }}
</el-tag>
```

### Q3: 什么时候可以填写物流？
**A:** 只有当 `auditStatus === 2`（已通过）且尚未填写物流时。
