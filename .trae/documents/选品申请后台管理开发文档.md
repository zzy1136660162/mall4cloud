# 选品申请功能 - 后台管理开发完整文档

## 📋 需求概述

### 业务流程

1. **普通用户申请成为达人** → 自动通过（在user_role表增加达人角色记录）
2. **达人申请选品商品进行邮寄** → 需要审核（审核通过后商家邮寄商品）
3. **选品申请表** → 即选品记录表，一张表同时记录申请和状态

### 功能清单

#### APP端（用户/达人）
1. ✅ 申请成为达人（自动通过）
2. ✅ 检查达人状态
3. ✅ 达人申请选品商品
4. ✅ 查看我的申请列表
5. ✅ 查看申请详情（含物流信息）

#### 管理端（商家/平台）
1. ✅ 查看选品申请列表（筛选：申请人、商品、审核状态、时间）
2. ✅ 审核选品申请（通过/拒绝）
3. ✅ 填写物流信息（快递公司、快递单号）
4. ✅ 查看统计数据（总数、待审核、已通过、已拒绝、今日新增）

## 📝 后端接口文档

### 1. APP端接口

#### 1.1 申请成为达人
- **URL:** `POST /app/talent/apply`
- **功能:** 用户申请成为达人，自动通过
- **请求参数:** 无
- **响应:**
```json
{
  "code": 200,
  "msg": "申请成功，您已成为达人",
  "data": null
}
```

#### 1.2 检查达人状态
- **URL:** `POST /app/talent/check`
- **功能:** 检查当前用户是否为达人
- **请求参数:** 无
- **响应:**
```json
{
  "code": 200,
  "data": true  // true=是达人，false=不是达人
}
```

#### 1.3 申请选品商品
- **URL:** `POST /app/selection/apply`
- **功能:** 达人申请选品商品
- **请求参数:**
```json
{
  "spuId": 123,                       // 商品ID（必填）
  "spuName": "商品名称",               // 商品名称（冗余）
  "shopId": 1,                        // 店铺ID（冗余）
  "shopName": "店铺名称",             // 店铺名称（冗余）
  "mainImgUrl": "https://xxx.jpg",   // 商品主图（冗余）
  "priceFee": 9999,                  // 商品价格（冗余）
  "contactPhone": "13800138000",      // 联系电话（必填）
  "deliveryAddress": "北京市朝阳区xxx", // 邮寄地址（必填）
  "applyRemark": "希望试用这个产品"     // 申请备注（可选）
}
```
- **响应:**
```json
{
  "code": 200,
  "msg": "申请成功，请等待审核",
  "data": {
    "applyId": 1
  }
}
```

#### 1.4 获取我的申请列表
- **URL:** `GET /app/selection/my-applies`
- **功能:** 获取当前达人的所有选品申请
- **请求参数:**
  - `pageNum`: 页码（默认1）
  - `pageSize`: 每页数量（默认10）
- **响应:**
```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "applyId": 1,
        "spuId": 123,
        "spuName": "商品名称",
        "mainImgUrl": "https://xxx.jpg",
        "priceFee": 9999,
        "shopName": "店铺名称",
        "auditStatus": 1,         // 1-待审核 2-已通过 3-已拒绝
        "auditStatusText": "待审核",
        "applyTime": "2026-04-11 10:00:00",
        "expressCompany": "顺丰速运",  // 审核通过后显示
        "expressNo": "SF123456789"      // 审核通过后显示
      }
    ],
    "total": 50,
    "pageNum": 1,
    "pageSize": 10
  }
}
```

### 2. 管理端接口

#### 2.1 获取申请列表
- **URL:** `GET /admin/selection/applies`

- **请求参数:**

  - `shopId`: 店铺ID（可选）
  - `auditStatus`: 审核状态（可选）
  - `startTime`: 开始时间（可选）
  - `endTime`: 结束时间（可选）
- **响应:**
```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "applyId": 1,
        "userId": 100,
        "userName": "张三",

        "spuName": "商品名称",

        "priceFee": 9999,
        "shopId": 1,
        "shopName": "店铺名称",
        "applyRemark": "希望试用",
        "applyTime": "2026-04-11 10:00:00",
        "auditStatus": 1,
        "auditStatusText": "待审核"
      }
    ],
    "total": 100,
    "pageNum": 1,
    "pageSize": 10
  }
}
```

#### 2.2 审核申请
- **URL:** `PUT /admin/selection/audit/{applyId}`
- **功能:** 审核选品申请
- **请求参数:**
```json
{
  "auditStatus": 2,              // 2-通过 3-拒绝
  "auditRemark": "审核通过"         // 审核备注（可选）
}
```
- **响应:**
```json
{
  "code": 200,
  "msg": "审核成功",
  "data": null
}
```

#### 2.3 填写物流信息
- **URL:** `PUT /admin/selection/ship/{applyId}`
- **功能:** 审核通过后，填写发货物流
- **请求参数:**
  - `expressCompany`: 快递公司（必填）
  - `expressNo`: 快递单号（必填）
- **响应:**
```json
{
  "code": 200,
  "msg": "发货信息已填写",
  "data": null
}
```

#### 2.4 获取统计数据
- **URL:** `GET /admin/selection/statistics`
- **功能:** 获取选品申请统计数据
- **响应:**
```json
{
  "code": 200,
  "data": {
    "totalApplies": 1000,       // 总申请数
    "pendingApplies": 50,         // 待审核数
    "approvedApplies": 900,      // 已通过数
    "rejectedApplies": 50,        // 已拒绝数
    "todayApplies": 20            // 今日新增
  }
}
```

### 第一阶段：API层开发（预计2小时）

#### 1.1 创建API文件
**文件:** `apps/example/src/api/modules/selection.ts`

```typescript
import api from '@/api'

// ============ 类型定义 ============

export interface SelectionApplyVO {
  applyId: number
  userId?: number
  userName?: string
  contactPhone: string
  deliveryAddress: string
  spuId: number
  spuName: string
  mainImgUrl: string
  priceFee: number
  shopId: number
  shopName: string
  applyRemark?: string
  applyTime: string
  auditStatus: number
  auditStatusText: string
  auditTime?: string
  auditRemark?: string
  expressCompany?: string
  expressNo?: string
  shipTime?: string
}

export interface SelectionApplyPageParam {
  pageNum?: number
  pageSize?: number
  userId?: number
  spuId?: number
  shopId?: number
  auditStatus?: number
  startTime?: string
  endTime?: string
}

export interface SelectionAuditParam {
  auditStatus: number
  auditRemark?: string
}

export interface SelectionStatisticsVO {
  totalApplies: number
  pendingApplies: number
  approvedApplies: number
  rejectedApplies: number
  todayApplies: number
}

// ============ API定义 ============

export const selectionApi = {
  // 获取申请列表
  page: (params: SelectionApplyPageParam) =>
    api.request({
      url: '/mall4cloud_platform/admin/selection/applies',
      method: 'get',
      params,
    }),

  // 审核申请
  audit: (applyId: number, data: SelectionAuditParam) =>
    api.request({
      url: `/mall4cloud_platform/admin/selection/audit/${applyId}`,
      method: 'put',
      data,
    }),

  // 填写物流信息
  ship: (applyId: number, expressCompany: string, expressNo: string) =>
    api.request({
      url: `/mall4cloud_platform/admin/selection/ship/${applyId}`,
      method: 'put',
      params: { expressCompany, expressNo },
    }),

  // 获取统计数据
  statistics: () =>
    api.request({
      url: '/mall4cloud_platform/admin/selection/statistics',
      method: 'get',
    }),
}

export default selectionApi
```

import { useRouter } from 'vue-router'
import { selectionApi } from '@/api/modules/selection'

const router = useRouter()

// 统计数据
const statistics = ref([])

// 查询参数

  pageNum: 1,
  pageSize: 10,
  auditStatus: undefined,
})

// 表格数据
const tableData = ref([])

  applyId: null as number | null,
  auditStatus: 2,
  auditRemark: '',
})

// 物流对话框
const shipDialogVisible = ref(false)
const shipForm = reactive({
  applyId: null as number | null,
  expressCompany: '',
  expressNo: '',
})

// 加载统计数据
const loadStatistics = async () => {
  const res = await selectionApi.statistics()
  statistics.value = [
    { label: '总申请数', value: res.data.totalApplies },
    { label: '待审核', value: res.data.pendingApplies },
    { label: '已通过', value: res.data.approvedApplies },
    { label: '已拒绝', value: res.data.rejectedApplies },
    { label: '今日新增', value: res.data.todayApplies },

  const res = await selectionApi.page(queryParams)
  tableData.value = res.data.list

  router.push(`/selection/detail/${row.applyId}`)

const submitAudit = async () => {
  if (!auditForm.applyId) return
  await selectionApi.audit(auditForm.applyId, {
    auditStatus: auditForm.auditStatus,
    auditRemark: auditForm.auditRemark,
  })
  ElMessage.success('审核成功')
  auditDialogVisible.value = false
  loadData()
  loadStatistics()
}

// 填写物流
const handleShip = (row: any) => {
  shipForm.applyId = row.applyId
  shipForm.expressCompany = ''
  shipForm.expressNo = ''
  shipDialogVisible.value = true
}

const submitShip = async () => {
  if (!shipForm.applyId) return
  await selectionApi.ship(
    shipForm.applyId,
    shipForm.expressCompany,
    shipForm.expressNo
  )
  ElMessage.success('物流信息已填写')
  shipDialogVisible.value = false
  loadData()
}

onMounted(() => {
  loadStatistics()
  loadData()
})
</script>

## ⚠️ 注意事项

### 1. API路径前缀
后端API路径前缀为 `/mall4cloud_platform`，需要确保前端请求路径正确。

### 2. 数据格式
- 价格：`priceFee` 单位为分，前端需要除以100转换为元
- 时间：`applyTime`、`auditTime` 等为日期时间字符串
- 状态：`auditStatus` 使用数字（1-待审核、2-已通过、3-已拒绝）

### 3. 权限控制
管理端接口需要登录认证和权限校验。

### 4. 图片显示
`mainImgUrl` 为商品主图URL，需要使用 `el-image` 组件显示。

### 5. 物流填写时机
只有审核状态为"已通过"且尚未填写物流时，才能填写物流信息。
