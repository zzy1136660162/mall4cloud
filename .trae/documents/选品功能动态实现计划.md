**文件路径**：`e:\ZZY_PROJECT\通用商城模板\common-mall\mall4cloud\front-end\mall4cloud-uniapp\src\utils\api\selection.js`

**创建内容**：

## 三、实施顺序

1. **创建 API 封装** (`src/utils/api/selection.js`) - 先创建，其他页面依赖此文件
2. **修改选品列表页** (`selection.vue`) - 核心页面，用户访问最多
3. **修改商品详情页** (`product.vue`) - 用户查看商品详情
4. **修改申请选品页** (`apply.vue`) - 用户提交申请
5. **修改我的申请列表页** (`list.vue`) - 用户查看申请记录
6. **修改样品详情页** (`detail.vue`) - 用户查看申请详情

## 四、注意事项

1. **错误处理**：每个 API 调用都需要 try-catch 处理
2. **加载状态**：显示 loading、结束 loading
3. **用户提示**：使用 `uni.showToast` 显示成功/失败提示
4. **价格转换**：后端价格单位是分，前端显示需要除以 100
5. **参数验证**：表单提交前必须验证必填项
6. **分页处理**：下拉刷新重置 page，上拉加载 page++
7. **页面跳转参数**：确保参数正确传递（spuId, applyId）
