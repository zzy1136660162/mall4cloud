# 旧商户管理平台迁移到新框架计划

## 一、项目概述

### 1.1 源项目 (`mall4cloud-multishop`)
* 技术栈：Vue 3 (JavaScript) + Element Plus + Vue Router + Pinia + Vite
* 功能模块：
  * 商品管理（商品列表、商品发布/编辑、商品分类、商品属性）
  * 订单管理（订单列表、订单详情、订单发货）
  * 店铺管理（热门搜索、首页图片）
  * 权限管理（角色管理）
  * 登录认证

### 1.2 目标项目 (`mall4cloud-multishop-refactor/apps/example`)
* 技术栈：Vue 3 + TypeScript + Element Plus + Vxe Table + Pinia + TailwindCSS
* 特点：动态路由、多种布局、权限控制、主题配置、标签页、面包屑导航
